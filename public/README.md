# Public Assets

Files in this folder are served directly without bundling.

## Current Contents
- `assets/icons/favicon.svg`: site favicon.
- `assets/video/student-decision-making-system.mp4`: primary intro video source.
- `assets/video/careerwiki-entry.mp4`: fallback intro video source.
- `assets/video/careerwiki-entry-poster.svg`: intro poster image.
- `robots.txt`: crawler directives (generated by SEO script).
- `sitemap.xml`: route map for indexing (generated by SEO script).

## Maintenance
- Keep only runtime-required static assets here.
- Regenerate `robots.txt` and `sitemap.xml` after route/data changes with `npm run seo:generate`.
